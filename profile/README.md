# Agostix 🚀

**Agostix** ist ein plattform- und hardwareagnostisches Deep-Learning-Framework, das von Grund auf neu entwickelt wurde. Es befreit Entwickler von den Abhängigkeiten schwerfälliger Ökosysteme und fehleranfälliger Treiberstrukturen wie ROCm unter Windows. 

Das Ziel von Agostix ist maximale, hardwarenahe Performance direkt auf der GPU – egal ob AMD (RDNA) oder NVIDIA (CUDA) im System steckt.

---

## 🔥 Die Philosophie

* **Kein ROCm-Frust unter Windows:** Agostix umgeht instabile Treiber-Stacks komplett durch den direkten Zugriff auf hardwarenahe Grafik-APIs
* **Echte Hardware-Unabhängigkeit:** Das Framework schert sich nicht um Markennamen. Es nutzt die rohe Rechenleistung moderner Grafikkarten (optimiert für die neuesten Generationen wie AMD RX 9000er RDNA-4 und NVIDIA-Counterparts)
* **Zero Dependencies (From Scratch):** Keine vorgefertigten Vorlagen, kein PyTorch- oder JAX-Überbau. Jede Matrix-Multiplikation und jeder Autograd-Graph ist selbst geschrieben.
